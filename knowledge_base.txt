# ProDevX Knowledge Base

## General Information

### What is ProDevX?
ProDevX is a platform for developers to showcase their projects, share blog posts, and connect with other developers. It's built with Next.js, TypeScript, and Convex backend.

### Tech Stack
- Frontend: Next.js 13+ with App Router
- Language: TypeScript
- Styling: Tailwind CSS with Shadcn UI components
- Backend: Convex
- Authentication: Built-in auth system
- Rich Text Editor: TipTap
- Form Handling: React Hook Form with Zod validation

### How to Set Up the Project?
1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables:
   - Configure Convex credentials
   - Set up authentication settings
4. Run development server: `npm run dev`
5. Access the application at http://localhost:3000

## Authentication

### How do I sign up?
1. Click on "Sign Up" in the navigation bar
2. Fill in your details:
   - Name
   - Email
   - Password
3. Submit the form to create your account

### How do I sign in?
1. Click on "Sign In" in the navigation bar
2. Enter your email and password
3. Click "Sign In" to access your account

### Common Auth Issues
1. Email verification not received
   - Check spam folder
   - Request new verification email
2. Password reset
   - Use "Forgot Password" link
   - Follow email instructions
3. Session expired
   - Re-login required
   - Check browser cookies

## Blog Features

### How do I create a blog post?
1. Navigate to "/blogs/create"
2. Fill in the blog details:
   - Title
   - Content using rich text editor
   - Add images (optional)
3. Click "Publish" to make your blog live

### Blog Management
- Edit blog: Click "Edit" on your blog post
- Delete blog: Use "Delete" option (only for your own blogs)
- Like/Unlike: Click heart icon
- Comment: Use comment section below blog posts

## Common Issues and Solutions

### Performance Issues
1. Slow loading:
   - Clear browser cache
   - Check internet connection
   - Verify server status

### Image Upload Problems
1. Image upload fails:
   - Check file size (max 5MB)
   - Supported formats: PNG, JPG, JPEG
   - Try compressing the image

### Error Messages
1. "Unable to connect to server":
   - Check internet connection
   - Server might be down
   - Clear browser cache

2. "Unauthorized access":
   - Login again
   - Check permissions
   - Session might have expired
